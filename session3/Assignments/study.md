## Questions:
- What is nested list?
- Can a list store both integers and strings in it?

## Anwers:

- Nested list is a list that contains another list
> ["cat", ["yellow", 18], 3000] 
 - A list can have any number of items and they may be of different types (integer, float, string etc.).
 > ["Tuan Anh", 25, 098]
